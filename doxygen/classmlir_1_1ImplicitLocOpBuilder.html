Detailed Description

ImplicitLocOpBuilder maintains a 'current location', allowing use of the create<> method without specifying the location.

It is otherwise the same as OpBuilder.

Create an operation of specific op type at the current insertion point, and immediately try to fold it.

This functions populates 'results' with the results of the operation.

Note: This performs opportunistic eager folding during IR construction. The folders are designed to operate efficiently on canonical IR, which this API does not enforce. Complete folding isn't only expected in the context of canonicalization which intertwine folders with pattern rewrites until fixed-point.
