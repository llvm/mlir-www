◆ nameToAttribute

llvm::DenseMap<StringRef, AbstractAttribute *> mlir::MLIRContextImpl::nameToAttribute

This is a mapping from attribute name to the abstract attribute describing it.

It is used by AbstractType::lookup to get an AbstractType from a name. As this map needs to be populated before StringAttr is loaded, we cannot use StringAttr as the key. The context does not take ownership of the key, so the StringRef must outlive the context.
