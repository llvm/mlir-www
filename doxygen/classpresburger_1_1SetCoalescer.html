Detailed Description

The SetCoalescer class contains all functionality concerning the coalesce heuristic.

It is built from a PresburgerRelation and has the coalesce() function as its main API. The coalesce heuristic simplifies the representation of a PresburgerRelation. In particular, it removes all disjuncts which are subsets of other disjuncts in the union and it combines sets that overlap and can be combined in a convex way.
