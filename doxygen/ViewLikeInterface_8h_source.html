48 /// Printer hook for custom directive in assemblyFormat.
49 ///
50 /// custom<OperandsOrIntegersOffsetsOrStridesList>($values, $integers)
51 ///
52 /// where `values` is of ODS type `Variadic<Index>` and `integers` is of ODS
53 /// type `I64ArrayAttr`. for use in in assemblyFormat. Prints a list with
54 /// either (1) the static integer value in `integers` if the value is
55 /// ShapedType::kDynamicStrideOrOffset or (2) the next value otherwise. This
56 /// allows idiomatic printing of mixed value and integer attributes in a
57 /// list. E.g. `[%arg0, 7, 42, %arg42]`.

59  Operation *op,

76 /// Pasrer hook for custom directive in assemblyFormat.
77 ///
78 /// custom<OperandsOrIntegersOffsetsOrStridesList>($values, $integers)
79 ///
80 /// where `values` is of ODS type `Variadic<Index>` and `integers` is of ODS
81 /// type `I64ArrayAttr`. for use in in assemblyFormat. Parse a mixed list with
82 /// either (1) static integer values or (2) SSA values. Fill `integers` with
83 /// the integer ArrayAttr, where ShapedType::kDynamicStrideOrOffset encodes the
84 /// position of SSA values. Add the parsed SSA values to `values` in-order.
85 //
86 /// E.g. after parsing "[%arg0, 7, 42, %arg42]":
87 /// 1. `result` is filled with the i64 ArrayAttr "[`dynVal`, 7, 42, `dynVal`]"
88 /// 2. `ssa` is filled with "[%arg0, %arg1]".
89 ParseResult parseOperandsOrIntegersOffsetsOrStridesList(
90  OpAsmParser &parser, SmallVectorImpl<OpAsmParser::OperandType> &values,
