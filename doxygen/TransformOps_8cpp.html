◆ matchBlock()

static DiagnosedSilenceableFailure matchBlock ( Block &  block,
ArrayRef< SmallVector< transform::MappedValue >>  blockArgumentMapping,
transform::TransformState &  state,
SmallVectorImpl< SmallVector< transform::MappedValue >> &  mappings 
)
static

Applies matcher operations from the given block using blockArgumentMapping to initialize block arguments.

Updates state accordingly. If any of the matcher produces a silenceable failure, discards it (printing the content to the debug output stream) and returns failure. If any of the matchers produces a definite failure, reports it and returns failure. If all matchers in the block succeed, populates mappings with the payload entities associated with the block terminator operands. Note that mappings will be cleared before that.
