mlir.dialects.linalg.opdsl.lang.types¶

Facility for symbolically referencing type variables.

Type variables are instances of the TypeVar class, which is uniqued by name. An “expando” accessor TV is provided that generates a named TypeVar for any attribute access:

TV.T TypeVar(T) TV.T is TV.U False TV.T is TV.T True
